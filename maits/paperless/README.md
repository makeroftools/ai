## About

@MAIT:#Paperless is the subject matter expert for Paperless-ngx, our self-hosted
document management system. Paperless provides document ingestion, AI-powered
auto-tagging, full-text search, and a native data connector into AnythingLLM for
RAG-powered document Q&A.

PRJ-013 (Paperless-ngx) is **✅ APPROVED** — this is the designated document
management hub for the WeOwn ecosystem.
